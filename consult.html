ホーム ／ 開発の相談をする

データセンター開発の相談を、
ここから。

土地の情報一枚から、事業化の可能性を一緒に検討します。構想段階でも、他社と検討中でも構いません。まずはあなたの立場に合う窓口をお選びください。

初回相談は無料です
GUIDE ｜ ご相談の立場について

どの立場からのご相談ですか

需要・供給・資金・連携・行政——それぞれの立場に合わせて、最適な窓口がご相談を承ります。ご自身に近いものをご確認のうえ、下のフォームの「ご相談の立場」からお選びください。

◆ つくりたい・使いたい（需要側）
TYPE 01

土地・不動産をお持ちの方

遊休地・工場跡地・変電所近接地などの活用可能性を知りたい。まず事業化できる土地かどうかを評価してほしい。

例：閉鎖工場の跡地、山林・原野、遊休農地
TYPE 02

DCを開発・運営したい方

データセンターを新たに開発・運営したい。立地評価・電力・許認可・資金・機材まで、開発全体を相談したい。

例：新規参入企業、既存事業者の増設
TYPE 03

計算力を売買したい方

空きGPUの計算力を売りたい運営者、または計算力を調達したいAI・SaaS・研究機関。取引板での売買を相談したい。

例：DC運営者、AI開発企業、研究機関
◆ 担い手として関わりたい（供給側）
TYPE 04

案件の担い手として参加したい事業者

EPC・冷却・電源・通信・運用などの担い手として、案件に参加したい。パートナー会員制度について知りたい。

例：ゼネコン・EPC、冷却設備、受変電、保守
TYPE 05

投資・出資を検討したい方

データセンター事業への投資に関心がある。サーバー分割投資（小口）や、案件への出資（大口）を検討したい。

例：事業法人、投資家、ファンド、富裕層
TYPE 06

本事業に参画・提携したい企業

プラットフォーム事業そのものへの参画・業務提携・共同事業・コンソーシアム参加を検討したい。

例：電力・通信・建設・IT大手、事業会社
◆ 地域から（行政）
TYPE 07

自治体・誘致担当の方

データセンターの誘致で地域を活性化したい。管内の電力・候補地・支援策の整理や、事業者との橋渡しを相談したい。

例：自治体、地域振興公社、地域金融機関

FORM ｜ ご相談内容の入力

相談フォーム

事業への参画・提携のご相談について：本件は運営代表が直接拝見します。差し支えなければ、ご提案の趣旨・想定されるご協力の形を「ご相談内容」欄にご記入ください。

送信後、担当より折り返しご連絡します。初回相談は無料です。
※このフォームは設置準備中です。公開時に送信先（お問い合わせ窓口）を接続します。

運営体制と中立性について

本相談窓口の運営は、一般社団法人グリーンXアライアンスが中立の立場で担っています。具体的な支援業務（コンサルティング・仲介の実務）は、グリーンX合同会社が受託します。

運営者およびその関係先が関与する案件については、その旨を必ず明示します。特定の事業者に偏らない、公正な立場でのご相談をお約束します。

費用について

情報の閲覧、そして初回のご相談は無料です。まずはお気軽にお問い合わせください。

その後、具体的な支援業務（立地評価・許認可・資金調達・補助金申請など）に進む場合のみ、着手金と成功報酬の形でご相談させていただきます。金額は案件の内容に応じて個別にご提示します。

ご相談の前に

よくいただく不安

まだ構想段階で、何も決まっていなくても相談できますか？
もちろんです。「土地があるが何ができるか分からない」「関心はあるが知識がない」という段階こそ、最もお役に立てます。決まっていないことを整理するところから、ご一緒します。
他社と相見積もり・並行検討をしていても大丈夫ですか？
問題ありません。むしろ複数の選択肢を比較検討されることをお勧めしています。中立の立場から、判断に必要な情報をご提供します。
相談したら、契約を勧められませんか？
無理な営業はいたしません。初回相談は、実現可能性を一緒に見極める場です。ご支援が必要と双方が判断した場合にのみ、次のステップをご提案します。
提供した土地や事業の情報が外部に漏れませんか？
ご相談内容は厳重に管理し、対応の目的以外には利用しません。必要に応じて秘密保持契約（NDA）の締結にも対応します。
